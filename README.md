# gulp-base-cli
基本gulp简易页面工具（适用于非SPA页面，可用于压缩js、css代码）
###根目录下文档结构
```
dist>   
     css>        
          min>    
                *.min.css   
          *.css   
     js>       
          min>   
                *.min.js    
          *.js   
src>     
     css>         
          *.styl    
     js>      
          *.js     
     images>      
          *.jpg|...    
     fonts>      
          ...    
```
